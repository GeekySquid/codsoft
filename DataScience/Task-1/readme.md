## 🌊 Project Overview

> *"Can we predict who survived one of history's most famous shipwrecks using Machine Learning?"*

This project applies **Machine Learning Classification** on the famous **Titanic Dataset** to predict whether a passenger survived based on personal and travel information.

It demonstrates the complete ML pipeline from raw data to prediction.

# 📂 Dataset Features

| Feature        | Description         |
| -------------- | ------------------- |
| 🎟 PassengerId | Unique Passenger ID |
| 🚢 Pclass      | Ticket Class        |
| 👤 Name        | Passenger Name      |
| 🚻 Sex         | Male/Female         |
| 🎂 Age         | Passenger Age       |
| 👨‍👩‍👧 SibSp | Siblings/Spouses    |
| 👶 Parch       | Parents/Children    |
| 🎫 Ticket      | Ticket Number       |
| 💰 Fare        | Ticket Fare         |
| 🏠 Cabin       | Cabin Number        |
| 🌍 Embarked    | Boarding Port       |
| ❤️ Survived    | Target Variable     |
